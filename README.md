# vuejs-pomodoro-app
Sample timer app made while following vue.js tutorial from Freecodecamp
## Project setup
```
yarn install
```

### Compiles and hot-reloads for development
```
yarn serve
```

### Compiles and minifies for production
```
yarn build
```

### Lints and fixes files
```
yarn lint
```
